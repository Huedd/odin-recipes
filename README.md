
# Odin Recipes Project

## Introduction
This project is part of The Odin Project curriculum. It focuses on building a basic recipe website using HTML.

The website will consist of a main index page with links to different recipe pages.

## Skills Demonstrated
By completing this project, I will have demonstrated the following skills:

- Creating a basic HTML structure
- Working with headings, paragraphs, and lists
- Adding links between pages
- Organizing files and folders in a project
- Using Git and GitHub for version control

## Reflection (Optional)
This project helps build a strong foundation in HTML and introduces the workflow of managing projects using Git.
